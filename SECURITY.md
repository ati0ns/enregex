# Security Policy

## Supported Versions

| Version  | Supported          |
| -------- | :----------------: |
| 1.8.x    | :white_check_mark: |
| <= 1.7.x | :x:                |

## Reporting a Vulnerability

To report a vulnerability you can go to the [Issues section](https://github.com/ati0ns/enregex/issues) or [Discord support server](https://discord.gg/dCFUy9cETt).
